# Churn-to-EBITDA-Impact-Model
## Overview
Int the telecom industrie losing a customer (**Churn**) is incredibly expensive because of the high initial **CapEx** spent to connect them. In this project I will be building a `Random Forest` model to predict customer churn
## Resources
- **The Data:** This project uses the [Telco Customer Churn Dataset]([https://www.kaggle.com/datasets/blastchar/telco-customer-churn] from Kaggle
